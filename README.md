# EON - Minimalist Fashion Website

A sleek, black and white fashion website for the EON brand, featuring a minimalist design aesthetic with modern styling.

## Features

- Clean, responsive design with black and white color scheme
- Modern typography with Montserrat and Playfair Display fonts
- Interactive elements with smooth animations
- Fully responsive layout for all device sizes
- Product showcase with hover effects
- Newsletter subscription functionality

## Technologies Used

- HTML5
- CSS3 (Flexbox and Grid layouts)
- JavaScript (ES6)
- Google Fonts

## Structure

- `index.html` - Main landing page
- `styles.css` - Custom styles and responsive design
- `script.js` - Interactive elements and animations
- `favicon.ico` - Brand favicon

## Design Elements

- Minimalist aesthetic with ample white space
- High contrast black and white color scheme
- Elegant typography hierarchy
- Subtle animations and transitions
- Clean product presentation

## Responsive Behavior

The website adapts to different screen sizes:
- Desktop: Multi-column layout with full navigation
- Tablet: Adjusted spacing and grid columns
- Mobile: Single column layout with hamburger menu

## How to View

Simply open `index.html` in any modern web browser.

## Customization

To modify the content:
1. Edit text in `index.html`
2. Adjust styles in `styles.css`
3. Modify interactions in `script.js`
